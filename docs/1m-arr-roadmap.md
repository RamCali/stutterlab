# StutterLab $1M ARR Roadmap

Target math examples:

- 1,000 customers at $99/month = $1.188M ARR
- 1,200 customers at $69/month = $994k ARR
- 800 customers at $99/month + SLP/B2B plans can exceed $1M ARR

## Product Thesis

StutterLab wins by becoming the most trusted daily speech practice system for people who stutter: structured treatment, emotionally safe AI practice, measurable progress, and responsible clinical framing.

## The Main Bottlenecks

1. Trust: billing must feel unquestionably legitimate.
2. First win: the first session must be completed successfully.
3. Voice quality: the AI must understand stuttered speech well enough to feel useful.
4. Retention: users need a clear next task every day.
5. Proof: progress reports must show meaningful change without overpromising.

## 30-Day Priorities

- Audit GTM and payment configuration.
- Improve first scenario completion.
- Run voice provider A/B test on real stuttering samples.
- Add cancellation/refund/support flow.
- Track first-session completion and trial activation.
- Replace open-ended scenarios with structured task flows.

## 60-Day Priorities

- Add a reliable daily plan loop.
- Add weekly progress emails.
- Add provider cost dashboard.
- Add guided exposure ladder progression.
- Add user interview loop for churned trials.
- Validate clinical language with an SLP advisor.

## 90-Day Priorities

- Launch SLP-reviewed progress reports.
- Add referral/testimonial engine after verified success.
- Add SLP/B2B plan exploration.
- Add real acoustic scoring benchmark.
- Add mobile voice QA matrix.
- Run paid acquisition only after first-session completion is stable.

## Key Metrics

- Visitor to signup
- Signup to checkout started
- Checkout started to trial activated
- Trial activated to first task completed
- First task completed to day-2 return
- Trial to paid
- Paid month-1 retention
- Voice session failure rate
- Manual fallback usage rate
- Support tickets per 100 trials

## Hard Rule

Do not scale marketing spend until first-session completion and billing trust are excellent. Paid traffic will amplify the current user experience, good or bad.
